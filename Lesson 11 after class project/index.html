ROBLOX


Roblox is an online game platform and game creation system developed by Roblox Corporation that allows users to program and play games created by themselves or other users.

Roblox was created by David Baszucki and Erik Cassel in 2004, and released to the public in 2006.

The platform hosts millions of user-created games (officially referred to as "experiences"), all created using a dialect of the programming language Lua and the platform's game engine, Roblox Studio.
